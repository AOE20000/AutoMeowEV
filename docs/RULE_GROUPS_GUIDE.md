# 配置文件使用指南

## 什么是配置文件？

配置文件是 MeowParser 2.1.0 引入的新功能，允许你创建多个独立的替换规则集合，并可以根据不同场景灵活启用或禁用。

---

## 为什么需要配置文件？

### 使用场景

1. **不同聊天场景**
   - 工作群组：使用正式的替换规则
   - 朋友群组：使用轻松的替换规则
   - 游戏频道：使用游戏相关的替换规则

2. **不同语言环境**
   - 中文配置文件
   - 英文配置文件
   - 混合配置文件

3. **测试和生产**
   - 测试配置文件：用于测试新规则
   - 生产配置文件：稳定的日常使用规则

---

## 如何使用配置文件？

### 创建配置文件

1. 右键托盘图标 → 替换规则
2. 点击新建"配置文件"按钮
3. 输入配置文件名称（如"工作场景"）
4. 点击确定

### 管理配置文件

- **重命名**：选择配置文件 → 点击"重命名"按钮
- **删除**：选择配置文件 → 点击"配置文件"按钮
- **切换**：点击配置文件名称切换到另一个配置文件

### 添加规则

1. 切换到可用配置文件
2. 右键规则组
3. 输入匹配文本和替换文本
4. 选择是否使用正则表达式
5. 点击确定

### 调整规则顺序

- 选择规则/规则组 → 右键选择"上移"或"下移"
- 规则按从上到下的顺序依次应用
- 规则组只起到折叠作用

### 示例

假设有以下配置：

```
规则组A（启用）：
  - "我" → "喵"
  
规则组B（禁用）：
  - "你" → "喵"
  
喵语转换（启用）：
  - 在标点前添加"喵"
  - 在句尾添加"喵"
```

输入："我和你不知道。"

执行过程：
1. 应用规则组A："喵和你不知道。"
2. 跳过规则组B（禁用）
3. 应用喵语转换："喵和你不知道喵。"
