# AutoMeow 自动发喵程序

一个简单的Windows系统托盘应用，可以自动在按下回车键时进行文本替换或发送"喵"。

## 功能特点

- 系统托盘运行，占用资源少
- 红/绿图标显示启用状态
- 一键开关功能
- 全局键盘监听
- 单实例运行保护
- 窗口白名单管理
- 智能文本替换功能（支持全文匹配替换）

## 使用方法

1. 运行程序后会在系统托盘显示一个"喵"字图标
2. 图标颜色表示状态：
   - 红色：禁用状态
   - 绿色：启用状态
3. 右键点击图标可以：
   - 启用/禁用功能
   - 打开窗口管理（选择启用的窗口）
   - 打开替换规则（配置文本替换）
   - 查看权限状态
   - 退出程序
4. 在窗口管理中：
   - 可以搜索窗口
   - 点击窗口切换启用状态
   - 可随时刷新窗口列表
5. 在替换规则中：
   - 可以添加、编辑、删除替换规则
   - 支持调整规则优先级（上移/下移）
   - 可以启用/禁用整个替换功能
   - 规则按顺序匹配，匹配到的文本会被替换
6. 权限状态显示：
   - ✓ 管理员：可以正常使用所有功能
   - ✗ 普通用户：功能受限，需要以管理员身份运行

## 文本替换功能

当按下回车键时：
- 程序会扫描当前行文本，将所有匹配的词替换为指定内容
- 支持多个规则同时生效，按优先级顺序依次替换
- 如果没有匹配到任何规则，会在句尾添加"喵"（原有功能）
- 每个规则会替换文本中所有匹配的位置

示例：
- 原句："这是我的一句话，他即将被AutoMeow替换"
- 规则："我" → "喵"，"他" → "喵"
- 结果："这是喵的一句话，喵即将被AutoMeow替换"

常用规则示例：
- "我" → "喵"
- "你" → "喵"
- "他" → "喵"
- "她" → "喵"

## 注意事项

- 需要管理员权限运行（因为需要键盘监听）
- 启用状态下会拦截Enter键
- 建议在聊天软件等适当场合使用
- 替换功能使用剪贴板，可能会短暂影响剪贴板内容

## 开发相关

如需从源码构建：

```bash
pip install -r requirements.txt
pyinstaller --clean --noconsole --onefile auto_meow.py
```

## 系统要求

- Windows 操作系统
- 不需要Python环境（使用打包版本）
