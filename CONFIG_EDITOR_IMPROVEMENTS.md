## 使用说明

### 基本操作

#### 1. 配置文件管理
```
顶部下拉框：选择配置文件
新建按钮：创建新配置
导入按钮：导入配置文件
导出按钮：导出当前配置
删除按钮：删除当前配置
```

#### 2. 规则组操作
```
添加规则组按钮：创建新规则组
双击规则组：切换折叠/展开
右键规则组：
  - 添加规则
  - 删除规则组
```

#### 3. 规则操作
```
双击规则：切换启用/禁用
右键规则：
  - 编辑规则
  - 上移
  - 下移
  - 删除规则
```

### 快捷操作

| 操作 | 方法 |
|------|------|
| 切换规则启用 | 双击规则 |
| 折叠规则组 | 双击规则组 |
| 添加规则 | 右键规则组 → 添加规则 |
| 编辑规则 | 右键规则 → 编辑规则 |
| 删除规则 | 右键规则 → 删除规则 |
| 调整顺序 | 右键规则 → 上移/下移 |
